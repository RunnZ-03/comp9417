\section{Bonus: Residual-Weighted AGOP}
\label{app:bonus}

\paragraph{(i) Justification.}
Standard AGOP weights all samples equally ($G_f = \tfrac{1}{n}\textstyle\sum_i \nabla f(x_i)\nabla f(x_i)^\top$).
Points with large residuals $r_i = y_i - f(x_i)$ indicate regions where the model fails.
Upweighting by $\varphi(r_i)=r_i^2$ shifts the AGOP toward directions of high model error,
providing a more focused signal for guided tree splits.

\paragraph{(ii) Implementation.}
The residual-weighted AGOP is defined as
\begin{equation}
  \mathrm{AGOP}_{\mathrm{res}}(f)
  = \frac{\displaystyle\sum_{i=1}^{n} r_i^2\,\nabla f(x_i)\nabla f(x_i)^\top}
         {\displaystyle\sum_{i=1}^{n} r_i^2}.
  \label{eq:agop-res}
\end{equation}
Gradients are estimated by forward finite differences ($\varepsilon=10^{-3}$) on a degree-2
Ridge regression model. The core implementation is two lines:
\begin{lstlisting}[language=Python, basicstyle=\ttfamily\small,
  frame=single, xleftmargin=4pt, xrightmargin=4pt, belowskip=2pt, aboveskip=2pt]
def agop(G, residuals=None):
    w = np.ones(len(G))/len(G) if residuals is None else residuals**2
    w /= w.sum() + 1e-12
    return np.einsum('i,ij,ik->jk', w, G, G)
\end{lstlisting}

\paragraph{(iii) Disagreement and Performance.}
On a two-regime dataset, residual-weighted AGOP identifies the signal in the hard-to-fit regime ($x_2$), whereas standard AGOP focuses on the well-fitted regime ($x_0$). Although both produce similar test RMSE in a single-level split, residual weighting provides a superior interpretive signal for identifying model failure regions.
