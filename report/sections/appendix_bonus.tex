\section{Bonus: Residual-Weighted AGOP}
\label{app:bonus}

\paragraph{(i) Justification.}
Standard AGOP weights all samples equally ($G_f = \tfrac{1}{n}\textstyle\sum_i \nabla f(x_i)\nabla f(x_i)^\top$).
Points with large residuals $r_i = y_i - f(x_i)$ indicate regions where the model fails.
Upweighting by $\varphi(r_i)=r_i^2$ shifts the AGOP toward directions of high model error,
providing a more focused signal for guided tree splits.

\paragraph{(ii) Implementation.}
The residual-weighted AGOP is defined as
\begin{equation}
  \mathrm{AGOP}_{\mathrm{res}}(f)
  = \frac{\displaystyle\sum_{i=1}^{n} r_i^2\,\nabla f(x_i)\nabla f(x_i)^\top}
         {\displaystyle\sum_{i=1}^{n} r_i^2}.
  \label{eq:agop-res}
\end{equation}
Gradients are estimated by forward finite differences ($\varepsilon=10^{-3}$) on a degree-2
Ridge regression model. The core implementation is two lines:
\begin{lstlisting}[language=Python, basicstyle=\ttfamily\small,
  frame=single, xleftmargin=4pt, xrightmargin=4pt, belowskip=2pt, aboveskip=2pt]
def agop(G, residuals=None):
    w = np.ones(len(G))/len(G) if residuals is None else residuals**2
    w /= w.sum() + 1e-12          # normalise; 1e-12 guards div-by-zero
    return np.einsum('i,ij,ik->jk', w, G, G)
\end{lstlisting}
On a single-regime dataset ($n{=}200$, $d{=}6$, $y=4x_0+3\sin(2x_1)+0.5\varepsilon$),
both AGOP variants recover $\{x_0,x_1\}$ as the top features
with cosine similarity $0.9994$, confirming correct implementation.

\paragraph{(iii) Disagreement.}
We construct a two-regime dataset ($n{=}600$, $d{=}6$) designed to expose the
difference between the two AGOP variants:
\begin{itemize}[nosep,leftmargin=1.5em]
  \item Regime A ($x_5<0$): $y=5x_0+0.05\varepsilon$ — well-fitted, residual RMSE $2.20$.
  \item Regime B ($x_5\ge 0$): $y=5x_2+5.0\varepsilon$ — poorly fitted, residual RMSE $5.10$.
\end{itemize}
A global Poly-2 Ridge model ($\alpha{=}0.5$) is fitted to the combined data.
Standard AGOP identifies $x_0$ (loading $0.863$) as the leading feature:
the well-fitted regime A dominates because its gradients are consistent.
Residual-weighted AGOP identifies $x_2$ (loading $0.682$): the large residuals
in regime B inflate the weights, shifting the leading direction toward the
hard-to-fit signal. The cosine similarity between the two leading eigenvectors
is $0.676$, confirming substantial directional disagreement ($p=1$).

\paragraph{(iv) Performance.}
Using the same two-regime DGP ($n_\text{train}{=}500$, $n_\text{test}{=}200$),
we compare RMSE of a one-level tree split using each AGOP direction:

\begin{center}\small
\begin{tabular}{lc}
  \toprule
  Method & Test RMSE \\
  \midrule
  Global model (no split)          & 4.034 \\
  Standard AGOP split (along $x_0$) & 4.151 \\
  Res-weighted AGOP split (along $x_2$) & 4.173 \\
  \bottomrule
\end{tabular}
\end{center}

Both single-level splits produce slightly higher RMSE than the global model,
and neither variant achieves a clear advantage over the other.
This suggests that in this controlled setting, residual weighting's main value
is \emph{interpretive}---correctly identifying where the model fails---rather than
automatically improving a single-split predictor.
Richer tree structures or iterative reweighting may be needed to realise
a performance benefit.
